Для докеризации требуется выполнение следующих команд:

1) mvn clean install -> docker build -t dangoose/fearandjoy:server .
2) docker run -d -p 28017:27017 --name joydb dangoose/fearandjoy:test_database
3) docker run -p 8082:8080 --link joydb dangoose/fearandjoy:server

Все образы расположены в Docker Hub: dangoose/fearandjoy.
